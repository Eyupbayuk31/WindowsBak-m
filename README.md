## BakimApp - Windows Bakım Uygulaması

Modern Windows 11 Fluent Design ile .NET 8 WPF bakım uygulaması.

### Özellikler
- **Dashboard** - CPU, RAM, Disk kullanımı, sistem bilgileri
- **Temizlik** - 8 kategori ile detaylı temizlik sistemi
  - Geçici Dosyalar
  - Prefetch Dosyaları
  - Geri Dönüşüm Kutusu
  - Windows Update Cache
  - Thumbnail Önbellek
  - DNS Önbellek
  - Son Dosyalar
  - Tarayıcı Önbellek (Chrome, Firefox, Edge)

### Gereksinimler
- Windows 10/11
- .NET 8 Runtime
- Yönetici yetkileri (otomatik istenir)

### Çalıştırma
```bash
dotnet run --project BakimApp
```
